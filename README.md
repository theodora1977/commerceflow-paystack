# CommerceFlow Paystack

CommerceFlow Paystack is a full-stack e-commerce transaction system that demonstrates a complete online checkout flow using the Paystack payment gateway. The project focuses on automated product data collection, dynamic product display, secure payment initialization, and server-side transaction verification.

This project was built as a collaborative team exercise where different teams handled Data Science, Backend Development, and Frontend Development.

---

## Project Overview

The goal of this project is to simulate a real-world e-commerce workflow where product data is collected automatically, displayed to users through a web interface, and purchased using secure online payments.

The system follows a simple pipeline:

1. **Data Collection** – Product information is gathered through web scraping.
2. **Product Display** – The frontend renders the product catalog for users.
3. **Checkout & Payment** – Users select products and pay using Paystack.
4. **Payment Verification** – The backend validates transactions and updates order status.

---

## Features

- Automated product data scraping
- Dynamic product catalog display
- Secure Paystack payment integration
- Payment initialization via backend API
- Webhook event handling for transaction confirmation
- Server-side payment verification
- End-to-end checkout flow
